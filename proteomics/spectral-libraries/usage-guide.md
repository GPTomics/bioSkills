# Spectral Libraries Usage Guide

## Overview

Spectral libraries contain reference MS2 spectra for peptides, enabling faster and more sensitive peptide identification in DIA and targeted proteomics.

## Library Types

### Empirical (DDA-based)
- Built from actual DDA experiments
- Highest quality spectra
- Limited to identified peptides
- Tools: SpectraST, EasyPQP, Skyline

### Predicted (Deep Learning)
- Generated by AI models
- Complete proteome coverage
- Slightly lower accuracy
- Tools: Prosit, MS2PIP, DeepLC

### Hybrid
- Combine empirical + predicted
- Best of both worlds
- Recommended for large-scale DIA

## When to Use

### Use Empirical Library
- Same sample type as library
- Targeted quantification
- High-confidence identifications needed

### Use Predicted Library
- New sample types
- Discovery proteomics
- Limited DDA data available

### Use Library-Free (DIA-NN)
- Quick analysis
- No existing library
- Exploratory studies

## Key Formats

| Format | Tool | Extension |
|--------|------|-----------|
| SpectraST | TPP | .splib |
| PQP | OpenMS/Skyline | .pqp |
| DLIB | EncyclopeDIA | .dlib |
| TSV | DIA-NN/OpenSWATH | .tsv |
| SSL | Spectronaut | .kit |

## Quality Metrics

- **Precursor count**: >50,000 for deep proteomics
- **Transitions per precursor**: 6-10 recommended
- **RT coverage**: Full gradient range
- **Charge distribution**: Mostly 2+ and 3+
